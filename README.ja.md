## 概要

Staqtapp-TDS
は、決定論的なネイティブコアを中心に設計された高性能ストレージエンジンです。

ネイティブストレージ処理、Python
インターフェース、ブラウザー運用コンソール、テレメトリ、および将来のネイティブインテリジェンス機能を明確に分離し、高い性能と保守性を両立しています。

## 主な特長

-   決定論的なネイティブストレージエンジン
-   Python 中心の開発インターフェース
-   ブラウザー運用コンソール
-   リアルタイムテレメトリ
-   ネイティブ診断
-   多言語対応
-   CPU ファースト設計
-   オプション機能のモジュール化

## バージョン 2.8.1

本リリースでは、Native Spiral Rank Engine
の実装に先立ち、管理ブラウザーのセキュリティ強化を実施しました。

### 主な改善

-   管理 POST 操作への CSRF 保護
-   Origin / Referer 検証
-   安全な DOM 描画への変更
-   リクエスト処理の堅牢性向上
-   ブラウザー言語設定の改善

ストレージエンジン本体および保存形式に変更はありません。

## 前回のリリース（v2.8.0）

-   英語、日本語、スペイン語、ポルトガル語、ドイツ語、フランス語、イタリア語
-   モジュール化された言語パック

## 設計思想

ストレージエンジンは常に決定論的な動作を維持します。

ブラウザー、テレメトリ、診断機能、および将来のインテリジェンスモジュールは、ストレージのホットパスから独立して動作します。
